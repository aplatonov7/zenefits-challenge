App is deployed on https://github.com/aplatonov7/zenefits-challenge

## To run the app locally:

1. npm install
2. npm run build
3. Open ./build/index.html in your browser of choice

## To run in dev mode:

1. npm install
2. npm start
3. Navigate to localhost:3000
